## POC Ecommerce NestJS with GraphQL

## Technologies
- NestJS
- MongoDB
- Mongoose
- Docker
- Docker compose
- JWT
- Passport
- GraphQL


## Installation

```bash
# Copy .env.example .env
$ cp .env.example .env
```

```bash
$ yarn install
```

## Running the app

```bash
# Run docker
$ docker compose up -d


# watch mode
$ yarn run start:dev

```
